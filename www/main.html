News

2007.03.10 Work has been progressing on the HUD - there are now gauges to show speed and health, ammo count, compass heading etc. This should lead to a simple mission structure, since the game can now show you where you need to go! Short video clip showing HUD . Needs Theora codec.

2007.01.28 Explosions! Now there is finally a reason to shoot at stuff :) (see new screen shot ).

2007.01.01 The planes can now smoke and burst into flames as they are damaged (see new screen shot ). AI is also progressing, it can now predict and avoid collisions with other planes.

2006.12.25 AI is getting better... planes can now (mostly) avoid (the rockiest bits of) mountains. They can also maintain distance from a target, and shoot at it accounting for it moving at a constant velocity, which makes them quite accurate. Next up: evasion, formation flying, and some higher level strategy to let them switch between behaviours.

2006.12.17 After a bit of a break from development, I've been cleaning the code up a little. There is now some simple AI steering for the planes, and a framework for adding more behaviour. Also auto-levelling for player controlled planes, and some performance improvements.

2006.08.27 Propeller effect added - see screenshot.

2006.08.16 Two new planes are finished - see screenshots.

 

Air Carrier

Air Carrier is an aerial combat game, aiming to support single and multiplayer, based around plane to plane dogfighting, and strategic use of airborne carriers.

Air Carrier is in development, but a playable version will be released as soon as possible. If you have some coding experience and would like to try out the current test level, please refer to the installation instructions.

The project is open source:
Code is available under the GPL licence.
Models and resources are available under a Creative Commons Attribution-NonCommercial-ShareAlike 2.0 License.

The game uses the 3D jMonkeyEngine, and models have been created using Blender.

About

The aim for Air Carrier is to produce models and an engine to support different game types based on aerial combat. This will be available for any use, but the initial focus will be on a fast paced, not particularly realistic dogfighting game, supporting deathmatch, team deathmatch, and hopefully other game types like CTF, etc.

Any help is very welcome. If you want to help, then please request an Observer role in the dev.java.net project. You can then use the Air Carrier project forums to ask me about Developer role, etc. Also, please check the #jme channel on freenode IRC if you want to discuss the project.
